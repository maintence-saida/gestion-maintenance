# Système de Gestion de Maintenance

Application web pour suivre la maintenance des équipements scolaires dans la wilaya El Bayadh.

## Fonctionnalités
- Chargement de fichiers Excel (format .xlsx)
- Visualisation des données par feuille
- Filtres par type d'établissement, technicien, statut
- Statistiques en temps réel
- Graphiques interactifs
- Export en CSV

## Technologies utilisées
- HTML5, CSS3, JavaScript
- SheetJS pour lire les fichiers Excel
- Chart.js pour les graphiques
- DataTables pour les tableaux interactifs

## Comment utiliser
1. Téléchargez ce projet
2. Ouvrez `index.html` dans votre navigateur
3. Chargez votre fichier Excel de maintenance
